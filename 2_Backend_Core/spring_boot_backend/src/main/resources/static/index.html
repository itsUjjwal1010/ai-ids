Live Attack Injectors
DDoS UDP Flood
Effect: Exhausts target's network bandwidth and CPU resources.
Mechanism: Sends a massive volume of spoofed UDP packets to random ports, forcing the target to repeatedly check for listening applications and reply with ICMP Destination Unreachable packets.
MITRE ATT&CK: T1498 (Network Denial of Service)
sec
DoS SYN Flood
Effect: Consumes target's connection queue, preventing legitimate TCP connections.
Mechanism: Rapidly initiates TCP connections by sending SYN packets but never completes the three-way handshake (no ACK), leaving connections "half-open" until they timeout.
MITRE ATT&CK: T1499 (Endpoint Denial of Service)
sec
Recon Port Scan
Effect: Identifies open ports, running services, and potential vulnerabilities.
Mechanism: Systematically probes a range of target ports by sending TCP SYN packets and analyzing the responses (SYN-ACK for open, RST for closed) to map the attack surface.
MITRE ATT&CK: T1595 (Active Scanning)
Mirai Botnet Sim
Effect: Simulates IoT botnet behavior, combining aggressive scanning and multi-vector flooding.
Mechanism: Emulates the Mirai malware by utilizing multiple threads to simultaneously launch high-volume floods (TCP/UDP) mimicking a coordinated botnet strike.
MITRE ATT&CK: T1059 (Command Interpreter) & T1498 (Network DoS)
sec
ARP Spoofing MITM
Effect: Intercepts and potentially alters traffic between the target and the gateway (Man-in-the-Middle).
Mechanism: Broadcasts forged ARP replies onto the local network, fraudulently associating the attacker's MAC address with the IP address of the default gateway.
MITRE ATT&CK: T1557 (Adversary-in-the-Middle)
